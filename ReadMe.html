 Service Method Url Description

.

 Product POST http://localhost:8080/DynamicWebPrj/ws/products Create a new product

.

 PUT http://localhost:8080/DynamicWebPrj/ws/products Update existing product

.

 GET http://localhost:8080/DynamicWebPrj/ws/products/{id} Retrive a Product for a given id

.

 GET http://localhost:8080/DynamicWebPrj/ws/products Retrive all products

.

 DELETE http://localhost:8080/DynamicWebPrj/ws/products/{id} Deletes a Product by given id

.

.

 Cart & CartItems POST http://localhost:8080/DynamicWebPrj/ws/carts Creates a new Cart and add the first item to it.

.

 POST http://localhost:8080/DynamicWebPrj/ws/carts/{id}/items Add a new Item the given cart id.

.

 PUT http://localhost:8080/DynamicWebPrj/ws/carts/{id} Update the cart.

.

 GET http://localhost:8080/DynamicWebPrj/ws/carts Retrive all carts

.

 GET http://localhost:8080/DynamicWebPrj/ws/carts/{id} Retrive a Cart for a given id.

.

 DELETE http://localhost:8080/DynamicWebPrj/ws/carts/{id} Deletes a cart by id

.

 DELETE http://localhost:8080/DynamicWebPrj/ws/carts/{cartid}/items/{itemid} Deletes an item from the given cart id
